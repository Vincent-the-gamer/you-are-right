# 序
你说得对，但是《你说得对，但是》是由乐子人自主传播的一系列全新网络舆论复读文本。内容传播于一个被称作「因特网」的数据空间，在这里，被大数据选中的人将被授予「文本框」，导引复读之力。你将扮演一位名为「键盘侠」的神秘角色，在持续的复读中邂逅情绪相近、思路独特的同伴们，和他们一起创造乐子，消解符号的严肃性--同时，逐步发掘「你说得对，但是」的真相。
